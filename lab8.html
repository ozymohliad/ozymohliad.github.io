Завдання 1.6

Створити HTML сторінку. На сторінці є рядок для введення тексту. Написати JavaScript-код. Якщо в межах рядка для введення клацнуто лівою кнопкою мишки і натиснуто кнопку Ctrl на клавіатурі – то в рядку для введення з’являється напис ‘1’. Якщо в межах рядка для введення клацнуто лівою кнопкою мишки і натиснуто кнопку Alt на клавіатурі – то в рядку для введення з’являється напис ‘2’. Якщо в межах рядка для введення клацнуто лівою кнопкою мишки і натиснуто кнопку Shift на клавіатурі – то в рядку для введення з’являється напис ‘3’.




Завдання 2.9
